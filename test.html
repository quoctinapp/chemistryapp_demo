Nhập chuỗi cần mã hóa
Cách sử dụng
+ File nhập vào cần phải có định dạng đuôi là .json
+ Cáu trúc file bên trong cần phải theo cấu trúc là 1 json
+ Nhập đầy đủ file và chuỗi cần mã hóa trước khi bấm nút "Done"
+ File json là file cho phép bạn tùy chỉnh dữ liệu để có kết quả mã hóa theo ý riêng
* Bạn có thể thao khảo file Json Tại đây
